# Space_Raiders--PUBLIC_Scripts
![logo_v2](https://github.com/Benja-Pauls/Space_Raiders--PUBLIC_Scripts/assets/73416124/c083ce24-7c81-4880-bfaa-e2624aaa9797)

Worked on part-time during school year, developed all game components ranging from scripts, models, UI, and advertising as a solo-dev venture. I made Space Raiders to gain additional development experience and release experience.<br/>
The premise of Space Raiders is an agario-style fighting game set in space where players fight and upgrade their ship as they gain additional materials. Random events around the map keep the game interesting and additional ship-customization is available as players gain coins collecting materials across different game sessions.
